# ushop-pc

## 项目介绍
```
1.本项目为练习测试项目，为pc端的商城管理页面，包含注册登录，以及对后台数据的添加，删除，修改，查看。
2.本项目后端代码使用node.js，前端代码采用Vue2框架编写，页面部分组件使用Element框架组件。
```
